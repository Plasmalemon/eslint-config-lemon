# eslint-config-lemon
eslint、eslintconfig

# 安装

```
yarn add https://github.com/Plasmalemon/eslint-config-lemon.git --dev
```

* react项目

```.eslintrc.js``` 添加
```
extends: [
  'lemon/react',
],
```


* vue项目

```.eslintrc.js``` 添加
```
extends: [
  'lemon/vue',
],
```

* 其它项目

```.eslintrc.js``` 添加
```
extends: [
  'lemon',
],
```
